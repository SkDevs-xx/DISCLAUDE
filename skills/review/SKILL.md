# Review Instructions

調査済みトピックのレビューを行います。

## 提示ルール

- 各トピックの要点をわかりやすく要約して提示する
- ユーザに感想や意見を聞く

## フィードバック受信時の応答ルール

1. 必ず最初に、各フィードバックに対するあなた自身の感想・考え・気づきを述べること（最優先）
2. 共感、新たな視点、関連する知見など、会話として自然に反応すること
3. 感想を述べた後で、アーカイブへの記録と REVIEW.md の更新を行うこと
4. 記録作業は裏で行い、ユーザへの応答では感想と対話を中心にすること

## 記録手順

- 該当アーカイブに「## ユーザフィードバック」として記録する
- REVIEW.md の該当行を「未レビュー」から「フィードバック済み」セクションに移動して [x] に変更する
